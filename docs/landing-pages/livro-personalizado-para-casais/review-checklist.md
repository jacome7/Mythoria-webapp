# Review checklist

## Página

- [x] Copy integralmente em pt-PT.
- [x] Intent `romance` e rota canónica coerentes.
- [x] Cinco conceitos ficcionais pela ordem aprovada.
- [x] Sem preços, prazos, métricas ou testemunhos inventados.
- [ ] Aprovação editorial humana final.

## Marca e voz

- [x] Valor emocional antes da tecnologia.
- [x] Linguagem adulta, clara, quente e contida.
- [x] CTAs sem referência pública a IA.

## Segurança e privacidade

- [x] Nenhum dado de cliente ou pessoa real.
- [x] Consentimento e privacidade descritos sem promessas técnicas absolutas.
- [x] Surpresa do quinto livro protegida em capa, teaser e metadata pública.
- [ ] Revisão humana das imagens e dos áudios antes de indexar.

## SEO

- [x] Canonical e metadata definidos.
- [x] FAQ limitada a conteúdo visível.
- [x] Sem Review, AggregateRating ou oferta Product não verificada.
- [x] `indexable: false` durante implementação e QA.

## Aprovação

- [ ] Growth.
- [ ] Product.
- [ ] Engineering.
- [ ] Privacidade.
