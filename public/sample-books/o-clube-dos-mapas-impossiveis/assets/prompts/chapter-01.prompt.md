# Chapter Image Prompt

Create an illustrated sample chapter image for "O Clube dos Mapas Impossíveis", a fictional Portuguese middle-grade workshop story for ages 10-12.

Landscape watercolor illustration. Show a library workshop room after closing, tables, bookshelves, pencils, folded maps, and a glowing hand-drawn map spreading across the floor. Three children collaborate around the map as tiny watercolor bridges, stairs, keys, and a lighthouse rise from the paper.

No text, no logos, no real library names, no watermark, no scary content.
