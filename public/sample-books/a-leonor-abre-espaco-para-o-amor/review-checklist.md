# Review checklist

- [x] Ficção e ausência de dados pessoais confirmadas.
- [x] Emoções mistas e papel voluntário da criança preservados.
- [x] Sem promessa de eliminar ciúmes, regressões ou conflito.
- [x] Estrutura familiar descrita sem estereótipos.
- [x] Capa, feature e título inspecionados após geração.
- [ ] Áudio ouvido integralmente em pt-PT.
- [ ] Aprovação editorial humana registada antes de publicação.
